# Code of Conduct

The Janus Governance Core project is committed to maintaining a respectful and collaborative environment.

## Expected behavior

Participants are expected to:

- communicate respectfully
- provide constructive feedback
- focus on technical clarity and governance integrity
- respect different perspectives and backgrounds

## Unacceptable behavior

The following behaviors are not tolerated:

- harassment or personal attacks
- discriminatory language
- disruption of technical discussions

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.

Repeated violations may result in exclusion from the project community.

## Scope

This Code of Conduct applies to:

- repository discussions
- pull requests
- issue tracking
- public project communication
