Janus Demos (Validation Artifacts)

Reproducible validation cases for Janus Core Lite

The Janus Demos (Validation Artifacts) provide deterministic and reproducible scenarios demonstrating the governance model implemented in Janus Core Lite.

Cases

  • Case 01 — Positive Evidence (E+) Demonstrates recording and validation of positive evidence.
  • Case 02 — Omission Detection (E−) Shows how protocol omissions are recorded as negative evidence.
  • Case 03 — Human Authority Override Illustrates how a human decision can validate or override protocol outcomes.
  • Case 04 — Schema Drift Reinterpretation Demonstrates reinterpretation when schema evolution occurs.
  • Case 05 — Stress Evaluation Tests protocol behavior under repeated or conflicting events.
  • Case 06 — Deterministic Rebuild Reconstructs system state from the append-only log.

Source Repository

All demos are reproducible directly from the repository.
